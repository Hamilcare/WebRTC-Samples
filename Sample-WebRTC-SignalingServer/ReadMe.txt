#WEB-RTC Signaling Server Sample
From [tutorialspoint](https://www.tutorialspoint.com/webrtc/webrtc_signaling.htm)

##Abstract
This is a quick implementation of a signaling server to use WebRTC running under nodejs and using webSocket.

##Setup environment
### Fedora 27
```
dnf install nodejs && 
npm install ws
```

##Launch
```
node server ./server.js
```
